Difference between document and window object in javascript

Window object:

It is the most object and outermost elements of the object hierarchy.

Document object:

Each HTML documents that gets loaded into a window becomes a document object.
The document contains the contents of the page.
using Document object, JavaScript can modify, add and delete the HTML elements, attributes CSS styles in the page

The window object represents a window/tab containing a DOM document
where as document object is property of window object
that points to the DOM document loaded in that window.

You can access a document object either using window.document property
or using document object directly as window is global object.
In the below example, title is the property of document object.

window.document.title returns the title of the page.

document.title also returns title of the page.

The other major difference is that both window object and document object have properties and methods.
Few method names are same in both objects but with different behavior.

window.open() opens a new tab or window and
document.open() creates a blank document within the window.
